September 13, 2025, Saturday, 10:26 PM Now I need to find a screen

It is a long task. I need to find a screen that checks all the items in the following list:

  • It should be small, like old point-and-shoot camera screens.
  • It should be a color display.
  • It should have decent resolution. I don’t need very high pixel density, but it should be good enough so that people’s faces are distinguishable.
  • It should be compatible with the Raspberry Pi Zero 2 W.
  • It shouldn’t consume much power.
  • It should be small enough that it doesn’t add much bulk to the camera. The camera should be slim enough to fit in my pants pocket.

I should find and order it soon, so that it arrives in time for my project week while I’m on leave.
